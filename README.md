### Requirements
- python 3.11
- npm 20.7.0
- docker 20

Backend API and admin panel are available on localhost:9080
The project is configured only for development!


### Initialization
```bash
make init
make prepare
```

### Local run
Run backend (available on localhost:9080):
```bash
make backend
```

Run frontend (available on localhost:9000):
```bash
make frontend
```
